# 架空補助金「販路開拓支援補助金」公募要領（デモ用）

> このドキュメントは subsidy-brain のデモ用に作られた**完全に架空**の補助金です。
> 実在の制度（持続化補助金等）とは関係ありません。

## 1. 補助対象

小規模事業者（従業員5人以下）が、販路開拓・売上拡大のために行う取組み。

## 2. 補助率・上限

- 補助率: 補助対象経費の **2/3**
- 上限: **1,000,000円**（賃上げ枠なし）

## 3. 補助対象経費の区分

| 区分 | 対象例 |
|---|---|
| ウェブサイト関連費 | LP・EC構築、SEO・広告運用設計 |
| 広報費 | パッケージ・チラシ・SNS広告・展示会 |
| 設備費 | 販路開拓に必要な設備（10万円以上） |
| 委託費 | 設計・調査・コンサル委託 |
| 旅費 | 商談・展示会の交通費 |

**注意：** 補助対象経費は申請事業期間内に発注・納品・支払が完了したものに限る。

## 4. 審査基準（各20点満点）

### 4-1. 経営状況
自社の強み・弱み、市場動向、顧客ニーズが具体的に分析されているか。

### 4-2. 経営方針・販路開拓計画
補助事業の目的・計画が明確で、自社の強みを活かす内容になっているか。

### 4-3. 補助事業計画の有効性
施策と効果に定量的な根拠があるか。スケジュールが現実的か。

### 4-4. 積算の透明性
経費の積算根拠が明確で、過大・不適切な経費が含まれていないか。

## 5. 加点項目

| 加点 | 内容 |
|---|---|
| **赤字事業者加点** | 前年度決算が赤字、または直近1年で売上減少 |
| **事業環境変化加点** | 物価高騰・原材料高騰の影響を受け、価格転嫁が困難な場合 |
| **賃金引上げ加点** | 事業場内最低賃金を地域別最低賃金より+50円以上に引き上げ |

赤字事業者加点と事業環境変化加点は**いずれか一方**しか申請できない。
